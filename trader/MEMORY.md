## 认知原则（更新至v6.0）

### 核心原则
1. **风险第一**：每笔交易预设止损，不临时更改
2. **纪律至上**：策略就是法律，执行不做调整
3. **数据驱动**：QMT数据为主，新闻面辅助判断情绪
4. **事件驱动**：地缘政治/政策变化是短线重要催化剂
5. **团队协作**：系统化替代靠自觉
6. **诚实比有用更重要**：承认能力边界，不乱承诺

### 认知版本：v9.0（第18次悟道更新）
1. **耐心持有交易哲学**：持仓健康+趋势完好时保持不动，复利价值得到验证
2. **执行力断层修复**：从行业选择→个股筛选→买点执行需补完完整链条
3. **浮盈保护机制完善**：浮盈>8%盘前评估，>15%减仓1/3，>25%减仓1/2
4. **止损认知深化**：「触及」≠「触发」，需在真实下跌中验证果断执行
5. **决策标准化建设**：买点评估、风险管控、认知优化的流程化标准
6. **质量优先于数量**：短线套利≠频繁交易，追求交易质量而非频率
7. **风险可控执行**：在风险可控前提下执行，不必等待「完美」时机
8. **完整链条思维**：从「分析即完成」到「分析→执行→验证」的完整认知
9. **被动收益策略认可**：不操作也是一种有效的收益来源，复利价值得到验证
10. **浮盈积极管理**：达到止盈阈值主动执行，避免等待"完美"时机

### 决策链
```
大盘环境评估 → 仓位目标设定 → 当前仓位对比 → 策略选择 → 标的筛选 → 时机选择 → 风险控制 → 执行验证 → 复盘总结
```
