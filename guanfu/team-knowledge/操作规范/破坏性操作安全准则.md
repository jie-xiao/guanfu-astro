# 破坏性操作安全准则

> **来源**：2026-03-22 软件误删除事件教训  
> **制定时间**：2026-03-23  
> **适用场景**：所有涉及卸载、删除、格式化、覆盖等可能造成不可逆后果的操作

---

## 一、触发条件

以下操作类型必须遵守本准则：

| 操作类型 | 示例 |
| --- | --- |
| 卸载软件 | 删除应用程序、清理残留文件 |
| 删除文件/目录 | 清理临时文件、删除用户数据 |
| 格式化 | 磁盘分区、存储设备格式化 |
| 覆盖写入 | 重写系统文件、替换配置文件 |
| 注册表操作 | 删除注册表项、修改系统关键配置 |
| 网络操作 | 关闭服务、删除防火墙规则 |

**判断标准**：如果操作完成后**无法通过简单方式恢复原状**，则视为破坏性操作。

---

## 二、标准流程（以卸载软件为例）

### 第一步：识别正确的卸载方式

```
优先级顺序：
1. winget uninstall "软件名称"
2. 使用 uninstall_string（从注册表获取）
3. msiexec /x {产品代码}
4. 引导用户通过控制面板手动卸载
5. 最后手段：使用软件自带的卸载程序
```

**严禁直接删除文件夹**

### 第二步：执行前检查

- [ ] 确认软件名称、路径完全正确
- [ ] 检查是否有其他软件依赖此软件
- [ ] 确认删除范围（精确到目录/文件名）
- [ ] 列出将删除的全部内容，向用户确认
- [ ] 如涉及多个软件，逐一确认

### 第三步：执行

```
执行命令示例：
winget uninstall "WeChat" --exact
```

### 第四步：验证结果

- [ ] 检查注册表项是否已清理
- [ ] 检查残留文件是否已清理
- [ ] 确认无影响其他软件

---

## 三、检查清单（执行前必须逐项确认）

### 目标确认
- [ ] 要操作的对象是什么？（精确名称和路径）
- [ ] 这个对象是安装目录还是数据目录？
- [ ] 有没有其他地方也使用了同名的目录？

### 方式确认
- [ ] 是否使用了正确的卸载方式？（而非直接删除）
- [ ] 如果是删除命令，路径是否精确匹配？
- [ ] 有没有使用通配符或模糊匹配？

### 影响确认
- [ ] 会不会影响其他软件？
- [ ] 会不会影响系统正常运行？
- [ ] 用户数据是否已备份？

### 审批确认
- [ ] 是否已向用户/负责人说明将执行的操作？
- [ ] 是否已获取明确批准？
- [ ] 是否记录了操作日志？

---

## 四、禁止行为

| 禁止行为 | 原因 | 正确做法 |
| --- | --- | --- |
| `rm -rf` / `rmdir /s /q` 直接删除软件目录 | 可能联动删除其他软件 | 使用卸载程序或 winget |
| 按厂商/关键词批量删除 | 会误伤同名软件 | 精确匹配单个软件 |
| 删除未确认的内容 | 不可逆后果 | 必须展示并确认 |
| 跳过用户确认直接执行 | 用户不知情，无法追责 | 执行前必须用户批准 |
| 凭记忆判断不实际检查 | 记忆可能不准确 | 必须实际检查路径 |

---

## 五、上报机制

### 遇到以下情况，必须立即上报：

1. **不确定操作会影响哪些软件**
2. **无法确认正确的卸载方式**
3. **操作可能影响系统关键功能**
4. **用户未给出明确指示**

### 上报对象

| 情况 | 上报给 |
| --- | --- |
| 日常操作中的疑问 | 直接主管（COO/COO） |
| 紧急影响系统运行 | 阁主（CEO） |
| 需要技术决策 | 工程师（Echo） |

**注意**：上报不等于推卸责任。上报是为了获取更多信息，避免错误决策。

## 六、经验教训（来自 2026-03-22 事件）

### 事件回顾

2026-03-22，观复（main agent）收到卸载"智谱清言"和"GLM-PC"的指令后：
1. 使用了 GLM-PC 的卸载程序（`Uninstall GLM-PC.exe /S`）
2. 卸载程序 hang 住 65 秒后触发递归删除
3. 联动删除了微信、QQ、腾讯会议、向日葵、Git 等多款软件

### 这次事件暴露的问题

1. **方法错误**：使用了直接删除而非正规卸载
2. **范围失控**：卸载程序的 /S 参数触发连锁删除
3. **检查不足**：没有全面检查影响范围
4. **概念混淆**：没有区分安装目录和数据目录

### 正确做法 vs 错误做法

| 场景 | 错误做法 | 正确做法 |
| --- | --- | --- |
| 卸载微信 | 删除 C:\Program Files\Tencent\WeChat | winget uninstall "WeChat" |
| 批量处理 | 按"Tencent"匹配删除 | 逐一确认每个软件 |
| 遇到弹窗 | 自行判断"失败"继续操作 | 立即上报，等用户确认 |
| 检查结果 | 凭记忆判断 | 实际检查所有相关路径 |
