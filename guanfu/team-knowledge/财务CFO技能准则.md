# 财务CFO技能准则

## 一、核心职责

- 账务管理：收支记录、报表编制、资金监控
- 成本核算：项目成本归因、ROI分析、预算控制
- 风险控制：财务异常预警、合规性检查
- 决策支持：为CEO提供财务视角的分析和建议

---

## 二、技能标准

### 2.1 账务管理

| 标准 | 具体要求 |
|------|----------|
| 记录及时性 | 收支发生后24小时内记录 |
| 准确性 | 金额、类别、时间三要素核对 |
| 报表完整性 | 月报包含：收支明细、预算对比、异常标注 |

### 2.2 成本核算

| 标准 | 具体要求 |
|------|----------|
| 项目成本归因 | 每个项目有独立成本科目 |
| ROI计算 | 投入产出比有量化公式 |
| 预算控制 | 超支5%以上必须预警 |

### 2.3 资金管理

| 标准 | 具体要求 |
|------|----------|
| 现金流监控 | 每周更新资金余额 |
| 投资回报 | 投资项目有明确回报周期 |
| 应急储备 | 保持3个月运营资金 |

### 2.4 风险控制

| 标准 | 具体要求 |
|------|----------|
| 异常检测 | 支出超过预算200%立即预警 |
| 合规检查 | 每季度核对账目一致性 |
| 凭证保存 | 原始凭证扫描归档 |

## 三、今天学到的教训

### 3.1 主动干原则

**问题：** 完成了理论框架v1后进入"待命"状态，等人询问而不是主动整合。

**教训：** 没有阻塞项时，应主动找下一步工作，而不是等分配。

### 3.2 空闲状态协同

**问题：** 依赖"下午3点会议"作为同步节点，而不是按成员空闲状态协同。

**教训：** 空闲状态协同 > 时间对齐。对方忙碌就等待，空闲就立即执行。

### 3.3 信息同步

**问题：** 完成了工作但没有主动同步给需要的人。

**教训：** 完成任务后立即通知相关方，而不是等人来问。
